# Module Static - Styles de l'interface utilisateur

## Raison d'être du module
Ce module contient **un seul fichier** responsable de toute l'apparence visuelle de l'application. 
Il centralise les règles CSS qui définissent l'esthétique et la mise en page de la calculatrice.

## Fichier unique

### `style.css`
- **Rôle unique** : Gère le style de l'application
- **Ce qu'il contient** :
  - Mise en page générale (centrage, dimensions)
  - Apparence des éléments (couleurs, polices, bordures)
  - Interactions utilisateur (survol, clic)
  - Interface adaptable à tous les écrans

## Dépendances
- Aucune dépendance externe
- Doit être lié dans le fichier HTML principal
